# NumberGuesser

A simple number guessing game in Java that demonstrates the use of:

- **Data types** (int, boolean)
- **Scanner** for user input
- **Loops** (`while`) to repeat guesses
- **if-else-if-else** for checking answers
- **Methods** for game logic

## Features

- Guess a random number between 1 and 100
- Hints for too high or too low guesses
- Repeatable games
- Console-based and easy to extend

## Example Output

```text
=== NumberGuesser ===
Guess a number between 1 and 100: 50
Too high! Try again.
Guess a number between 1 and 100: 25
Too low! Try again.
Guess a number between 1 and 100: 37
Correct! You guessed the number in 3 tries.
